# ⚡ NEXUS • AI OS

Welcome to **NEXUS AI OS** - A powerful, web-based multi-functional AI workspace and virtual operating system interface. This project integrates multiple utility tools, AI features, and APIs into a single, seamless, glass-morphism designed web application.

## 🚀 Features (১১টি পাওয়ারফুল টুল)

1. 🧠 **AI Chat:** Smart AI assistant for answering your daily queries.
2. 🌍 **Global Chat:** Live public chatroom (powered by Firebase Realtime Database).
3. 🎨 **Image Studio:** Generate stunning AI images directly from text prompts.
4. 🐍 **Python Lab:** Run Python code directly in the browser natively using Pyodide.
5. 💻 **Code Lab:** Write and preview HTML/CSS/JS code instantly.
6. ⛅ **Live Weather:** Check current live weather updates for any city (via wttr.in).
7. 🌐 **Web Browser:** In-app browser with external tab support.
8. 📚 **Library & Wiki:** Search books (Google Books, Open Library, Gutendex) & read Wikipedia articles.
9. ⛩️ **MangaDex:** Search your favorite manga and view their cover arts.
10. 🦆 **DuckDuckGo:** Private, tracking-free search engine integration.
11. 📸 **URL Scanner:** Capture live screenshots and scan any website URL instantly.

## 🛠️ Tech Stack

* **Frontend:** HTML5, Tailwind CSS (via CDN), Vanilla JavaScript (ES6+)
* **Database (Chat):** Firebase Realtime Database
* **Python Runtime:** Pyodide
* **External APIs:** 
  * WTTR.in (Weather)
  * Pollinations AI (Image Generation)
  * Google Books, Open Library, Gutendex (Books)
  * Wikipedia API (Information)
  * MangaDex API (Manga)
  * DuckDuckGo API (Search)
  * Thum.io (URL Screenshot)
